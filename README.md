## Quick Start

1. Install the plugin:

```bash
npm install opencode-im-bridge
```

2. Add to your `opencode.json`:

```json
{
  "plugin": ["opencode-im-bridge"]
}
```

3. Configure your IM bridge:

```bash
npx opencode-im-bridge setup
```
